# Amazon Prime TV Shows and Movies Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Amazon Prime dataset to understand content distribution, genre trends, and audience preferences.

## Objectives
- Analyze distribution of movies and TV shows
- Identify most common genres
- Study content growth over time
- Analyze IMDb ratings distribution
- Examine relationship between ratings and popularity

## Dataset
The dataset consists of:
- titles.csv (main dataset used)
- credits.csv (cast and crew data)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate the platform compared to TV shows
- Drama is the most common genre
- Content increased significantly after 2015
- Most IMDb ratings fall between 5 and 7
- Popularity is not directly dependent on ratings

## Conclusion
This project demonstrates how exploratory data analysis can be used to extract meaningful insights and understand trends in streaming platforms.
